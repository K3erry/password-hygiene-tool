# Password Hygiene Analytics Tool

## Overview
Browser extension for real-time password strength and breach exposure detection using k-anonymity.

## Features
- Real-time password strength analysis (zxcvbn)
- Breach checking via HaveIBeenPwned API
- 100% client-side processing
- Privacy-preserving k-anonymity

## Installation
1. Download this folder
2. Open Chrome → `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" → Select this folder
